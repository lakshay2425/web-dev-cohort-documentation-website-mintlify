# Mintlify Documentation Clone

This project is a desktop-first HTML & CSS clone of the [Mintlify](https://mintlify.com/) documentation website.

## Recreated Sections

The following 10 sections have been faithfully recreated to match the structural layout and aesthetics of the original site:

1. **Top Navigation Bar**: Logo, navigation links, and primary Call-To-Action buttons.
2. **Hero Section**: Main headline, description, email input with CTA, and the beautiful background illustration.
3. **Documentation Preview Section**: A complete mock of the Mintlify docs interface, including the sidebar navigation, top bar, main content, and table of contents.
4. **Trusted By / Logos**: A row highlighting companies using Mintlify.
5. **Feature Highlights**: Two-column alternating sections demonstrating key capabilities with mock terminal and UI elements.
6. **Intelligent Assistant / UI Preview**: A mockup of the AI assistant chat interface showcasing interactive capabilities.
7. **Enterprise Features Section**: A dedicated section highlighting enterprise support and compliance.
8. **Case Studies / Customer Stories**: A card-based layout featuring customer success metrics and stories.
9. **Final Call-To-Action**: A strong concluding section encouraging users to sign up.
10. **Footer**: Multi-column footer containing links to all areas of the site and legal information.

## Design System

### Fonts
The project exclusively uses **Inter**, the same primary geometric sans-serif typeface used by Mintlify, fetched via Google Fonts.

### Colors
The primary brand colors have been closely matched:
- **Mountain Meadow (Primary)**: `#18E299`
- **Woodsmoke (Background)**: `#08090A`
- **Text Primary (White)**: `#FFFFFF`
- **Text Secondary (Gray)**: `#A1A1AA`
- **Surface Backgrounds**: Variations of `#111111` and `#1A1A1A` with subtle white borders (`rgba(255, 255, 255, 0.1)`) for card and section separation.

## Running the Project
Simply open `index.html` in any modern web browser to view the project.
